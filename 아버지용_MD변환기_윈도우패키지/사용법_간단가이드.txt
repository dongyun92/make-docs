==========================================
      아버지용 MD → DOCX 변환기 사용법
==========================================

🎯 이 프로그램은 무엇인가요?
- 마크다운(MD) 파일을 워드 문서(DOCX)로 변환해줍니다
- HTML 차트들을 선택해서 문서에 이미지로 삽입할 수 있습니다
- 깔끔한 서식이 적용된 전문적인 워드 문서가 완성됩니다

📋 사용 순서

1단계: 프로그램 설치
   ✅ "setup_final.bat" 파일을 더블클릭
   ✅ 설치가 완료될 때까지 기다리기
   ✅ "Setup completed!" 메시지가 나오면 아무 키나 누르기

2단계: 프로그램 실행  
   ✅ "start_program.bat" 파일을 더블클릭
   ✅ 파란색 창이 나타날 때까지 기다리기

3단계: 파일 변환하기
   ✅ "Markdown 파일" 찾아보기 버튼 클릭 → MD 파일 선택
   ✅ 필요하면 "HTML 차트 파일들 선택" → 차트 추가
   ✅ "출력 폴더" 설정 (저장할 위치)
   ✅ "변환 시작" 버튼 클릭

4단계: 완성 확인
   ✅ 상태 창에서 진행상황 확인
   ✅ "변환 완료!" 메시지가 나오면 끝
   ✅ 지정한 폴더에서 DOCX 파일 확인

==========================================

📁 폴더 구조

📂 images/          ← HTML 차트 파일들 여기에 넣으세요
📂 output/          ← 완성된 DOCX 파일이 여기에 생성됩니다  
📂 template/        ← 참고용 템플릿과 가이드
📄 *.md            ← 변환할 마크다운 파일

==========================================

💡 사용 팁

✅ MD 파일 작성 시:
   - template/ 폴더의 가이드 참고
   - 이미지는 ![제목](images/파일명.png) 형식으로 작성

✅ 차트 추가 시:
   - images/ 폴더에 HTML 파일들 넣기
   - GUI에서 필요한 차트만 선택

✅ 문제 발생 시:
   - 상태 창에서 오류 메시지 확인
   - "로그 지우기" 버튼으로 화면 정리

==========================================

🛠️ 문제 해결

"Python을 찾을 수 없습니다"
   → Python 설치 후 PATH 설정 필요

"라이브러리 오류"  
   → setup_final.bat를 관리자 권한으로 다시 실행

"Chrome을 찾을 수 없습니다"
   → Google Chrome 브라우저 설치 필요

변환이 안될 때
   → MD 파일과 HTML 파일 경로 확인
   → 파일명에 특수문자 없는지 확인

==========================================

이 프로그램으로 마크다운 파일을 
전문적인 워드 문서로 쉽게 변환하세요! 📝
