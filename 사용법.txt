==========================================
     사업계획서 자동 생성 프로그램 사용법
==========================================

📁 설치 방법:
1. 압축파일을 원하는 폴더에 압축해제
2. setup_final.bat 실행 (관리자 권한 권장)
3. 모든 설치가 완료될 때까지 대기

🚀 프로그램 실행:
1. start_program.bat 실행
2. GUI 창이 열림
3. 마크다운 파일(.md) 선택
4. 출력 경로 및 파일명 입력
5. "변환 시작" 클릭

📝 마크다운 파일 작성법:
- template_example.md 파일을 참고하여 작성
- # 제목, ## 소제목, ### 세부제목 형식 사용
- 표는 | 구분자로 작성
- 차트는 ```chart 블록으로 작성

🔧 문제 해결:
- Chrome 미설치: https://www.google.com/chrome/ 에서 설치
- Python 미설치: https://www.python.org/downloads/ 에서 설치
  (설치 시 "Add Python to PATH" 반드시 체크)
- 권한 오류: 관리자 권한으로 setup_final.bat 재실행

⚠️ 주의사항:
- 바이러스 백신이 실행을 차단할 수 있음 (일시 해제 후 실행)
- 인터넷 연결 필요 (라이브러리 설치용)
- Windows 10 이상 권장
